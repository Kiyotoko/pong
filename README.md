|:-------|------------|:------------:|
| 1      | Move up    |     `W`      |
| 1      | Move down  |     `S`      |
| 2      | Move up    |  `ARROW UP`  |

## Run pong

> ⚠️ Please note that you need to install java on your machine first before you can run pong.

```shell
java -jar pong-*.*-uber.jar
```

## Build on your own

```shell
mvn install
```
